Project Description

The Blood Donation Portal is a web-based application developed to connect blood donors with recipients in an efficient and organized manner. The system helps users search for blood donors based on blood groups and manage blood donation requests during emergencies.

This project aims to simplify the blood donation process and promote awareness about voluntary blood donation through an easy-to-use digital platform.

🎯 Objectives
To provide a platform for blood donors and recipients.
To reduce the time required to find blood donors during emergencies.
To maintain donor and recipient records efficiently.
To encourage voluntary blood donation awareness.

✨ Features
User Registration and Login
Donor Registration System
Blood Group Search Functionality
Blood Request Management
Secure Authentication
Responsive User Interface
Emergency Blood Request Support
